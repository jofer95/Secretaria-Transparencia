Java JSON Client Library

Created 10 de abril de 2018

The Java client-side library is used to provide the set of Java objects that can be serialized to/from JSON using Jackson. This is useful for accessing the JSON REST endpoints that are published by this application.

Resources Example (Raw JAXB)
java.net.URL url = new java.net.URL(baseURL + "/accesos/login");
ObjectMapper mapper = new ObjectMapper();
java.net.URLConnection connection = url.openConnection();
connection.setDoOutput(true);
connection.connect();

mapper.writeValue(connection.getOutputStream(), modelLogin);
ResponseOfModelLogin result = (ResponseOfModelLogin) mapper.readValue( connection.getInputStream(), ResponseOfModelLogin.class );
//handle the result as needed...
Resources Example (Jersey client)
javax.ws.rs.client.Client client = javax.ws.rs.client.ClientBuilder.newClient();

ResponseOfModelLogin result = client.target(baseUrl + "/accesos/login")
  .post(javax.ws.rs.client.Entity.entity(modelLogin, "application/json"), ResponseOfModelLogin.class);

//handle the result as needed...
